# PropertyWebBuilder Documentation

## Overview

Welcome to the PropertyWebBuilder documentation. This documentation is intended to provide a comprehensive overview of the PropertyWebBuilder application, from its high-level architecture to the details of its individual components.

PropertyWebBuilder is a Ruby on Rails application for creating real estate websites. It is designed to be a flexible and extensible platform that can be used to create a variety of different types of real estate websites, from simple property listings to more complex, feature-rich applications.

This documentation is divided into several sections, each of which covers a different aspect of the application. The sections are as follows:

*   **Overview:** This section provides a high-level overview of the application, its architecture, and its main components.
*   **Data Models:** This section describes the application's data models, including their attributes, validations, and associations.
*   **Controllers:** This section describes the application's controllers, including their actions and how they interact with the models and views.
*   **API:** This section describes the application's API, including both the RESTful API and the GraphQL API.
*   **Frontend:** This section describes the application's frontend, including the Vue.js applications that are used to render the user interface.
